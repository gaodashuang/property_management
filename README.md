#### 在2020年春节期间做的一个物业管理系统，基于vue-cli4、vue、element和beego实现。

---

### 功能列表
- 首页
- 小区管理
- 房产信息管理
- 住户信息管理
- 停车位管理
- 收费管理
- 资产设备管理
- 值班管理
- 系统配置
